# GoTimr
A small CLI app that creates a timer in your terminal 
You can set a title, duration and update interval as flags
to GoTimr. Hoping to expand this with adding multiple
Timrs and a working menu.

## Why GoTimr?
I chose this as a first personal project because I wanted to challenges
myself in using language features that I don't often use like:
- goroutines
- channels

## Challenges
The most challenging part was getting the channel bits right. I've seen
a channel deadlock way more often than I dare to admit. But hell did I learn alot.
